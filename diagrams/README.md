# 📊 Architecture Diagrams

Source files for architecture diagrams used throughout the portfolio.

## Tools Used

- **Mermaid** — Inline diagrams in markdown (rendered by GitHub)
- **draw.io** — Complex architecture diagrams (XML source files)
- **Lucidchart** — Collaborative diagrams for presentations

## Diagram Standards

- Use Mermaid for simple flow and sequence diagrams
- Use draw.io for complex multi-service architectures
- Export PNG versions to `assets/images/` for non-GitHub contexts
- Include source files for editability
