## Pattern Template

Each pattern includes:
- **Overview** — What the pattern solves
- **Use Cases** — When to apply it
- **Architecture Diagram** — Mermaid diagram
- **Pros/Cons** — Trade-off analysis
- **Best Practices** — Implementation guidance
- **Anti-Patterns** — What to avoid
