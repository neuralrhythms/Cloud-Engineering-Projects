# ⚡ Lambda Automation

> Serverless event-driven automation for security remediation, cost optimization, and operations.

## Use Cases

| Function | Trigger | Action |
|----------|---------|--------|
| Security Auto-Remediation | EventBridge (GuardDuty) | Isolate compromised instances |
| Tag Compliance | Config Rule (non-compliant) | Apply default tags |
| Cost Alerts | Budget threshold | Notify + stop non-critical resources |
| Snapshot Cleanup | Scheduled (daily) | Delete unattached snapshots |
| SSL Certificate Rotation | ACM expiry event | Renew and deploy certificates |
