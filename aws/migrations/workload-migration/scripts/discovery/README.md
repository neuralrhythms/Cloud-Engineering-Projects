# Discovery Scripts

This directory contains scripts and utilities for the Phase 0 discovery activities.

## AWS Application Discovery Service (ADS)

### Agent Deployment

Deploy the ADS agent on VMware-hosted servers to collect:
- Server inventory (OS, CPU, memory, disk)
- Network connections and dependencies
- Running processes and installed software

### Pre-requisites

- AWS CLI configured with credentials for the Migration Hub account
- ADS agent installer downloaded from AWS console
- Network connectivity from SDDC servers to ADS endpoints
